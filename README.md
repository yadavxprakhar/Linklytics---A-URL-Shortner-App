# 🚀 Lynkforge — Full-Stack URL Shortener with Analytics

Lynkforge is a production-ready full-stack application that enables users to generate, manage, and track shortened URLs with secure authentication and real-time analytics.

> Built to demonstrate strong backend architecture, secure API design, and modern frontend development.

---

## 🌟 Key Highlights

- 🔐 JWT-based authentication & authorization  
- ⚙️ RESTful APIs built with Spring Boot  
- 📊 URL analytics & click tracking  
- ⚡ Responsive React frontend with smooth routing  
- 🧠 Scalable full-stack architecture  
- 🔒 Secure environment & configuration handling  

---

## 🛠️ Tech Stack

### Frontend
- React.js  
- Tailwind CSS  
- Axios  
- React Router  

### Backend
- Spring Boot  
- Spring Security  
- JWT Authentication  
- REST APIs  

### Database
- MySQL / PostgreSQL *(update if needed)*  

---

## 🚀 Features

- 🔗 Shorten long URLs  
- 👤 User authentication (Signup/Login)  
- 📈 Track link clicks and analytics  
- 📂 Manage URLs via dashboard  
- 🔄 Automatic redirection
